
# Web Log Analyzer

A Flask-based web application for analyzing log files.

## Features

- Keyword search
- Case-sensitive search
- Log level filtering
- File upload support
- Styled analysis results

### Excel Export Features

- History sheet with all saved searches
- Stats sheet with summary statistics
- Auto-sized columns
- Freeze panes for headers
- Colored header rows
- Zebra-striped rows in History sheet
- Excel filter dropdowns
- Most searched keyword statistic

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Git & GitHub
